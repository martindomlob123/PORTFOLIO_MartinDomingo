## About the project

This project is a front-end portfolio template focused on:

- Visual clarity
- Simple navigation
- Easy customization

It is suitable for designers, developers, and creatives who want a clean base to present their work online.

## Features

- Responsive layout
- Modular sections (About, Projects, Contact, etc.)
- Custom 404 error page
- Lightweight and fast loading
- Easy to personalize (colors, typography, content)

## Built with

- HTML5
- CSS3
- JavaScript (vanilla)

No frameworks required.
